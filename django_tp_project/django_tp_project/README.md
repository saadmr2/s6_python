# TP MI-GUIDE Django

Projet réalisé à partir du sujet:
- Installation de Python et Django
- Création d'un environnement virtuel
- Création du projet `myproject`
- Création de l'application `myapp`
- Configuration des URLs, vues et templates
- Compteur de mots
- Authentification: register, login, logout
- Modèle `Feature`, migrations et interface admin

## Commandes principales

```bash
python -m venv monenv
monenv\Scripts\activate
pip install -r requirements.txt
python manage.py migrate
python manage.py createsuperuser
python manage.py runserver
```

## Structure
- `myproject/`: configuration principale
- `myapp/`: application
- `templates/`: pages HTML
- `static/`: CSS
